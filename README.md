# Road Pothole Detection using YOLOv8

A Streamlit-based web application for pothole detection using YOLOv8 Instance Segmentation.

## Live Demo

[Click Here to Open App](https://yolov8-pothole-instance-segmentation.streamlit.app/)

## Features

- Image pothole detection
- Video pothole detection
- Area calculation
- Instance segmentation
- Confidence analytics
- Download processed output
- Mobile-friendly UI

## Tech Stack

- Python
- Streamlit
- YOLOv8
- OpenCV
- NumPy
